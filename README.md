# 📇 Smart Visiting Card Scanner

A professional AI-powered Visiting Card Scanner web application built using HTML, CSS, JavaScript, and Tesseract.js OCR.

This application uses OCR (Optical Character Recognition) to scan business/visiting cards through the camera, automatically extract important details, and save them locally for future access.

---

## 🚀 Features

- 📷 Live Camera Capture
- 🔍 OCR Text Recognition using Tesseract.js
- 👤 Automatic Name Detection
- 💼 Job Role / Designation Detection
- 🏢 Company Name Extraction
- 📞 Smart Phone Number Detection
- 📧 Email Detection
- 📍 Address & Pincode Extraction
- 🌐 Website Detection
- 💾 Local Storage Backup
- 📂 View Previously Scanned Cards
- 📱 Mobile Responsive UI
- 🎨 Professional User Interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Tesseract.js OCR
- LocalStorage API

---

## 📸 How It Works

1. Open the application
2. Allow camera permission
3. Show the visiting card in front of the camera
4. Click **Capture & Scan**
5. OCR extracts the card details automatically
6. Save the scanned data
7. View saved cards anytime using Backup
